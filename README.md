# Ecommerce-sales-analysis-dashboard
Excel-based e-commerce sales analysis project with KPI tracking, YoY comparison, and interactive dashboards.
Project Overview:
Developed an interactive E-commerce Sales Analysis Dashboard using MS Excel to analyze sales performance, profitability, and customer trends. The dashboard enables data-driven decision-making through dynamic visuals and KPIs.
Key Highlights:
•	Designed KPI dashboards to track Total Sales, Profit, Orders, and Quantity
•	Performed Year-over-Year (YoY) analysis to measure business growth
•	Built bar, pie, combo, waterfall, and map charts for multi-dimensional insights
•	Analyzed category, region, and time-based performance
•	Implemented slicers and interactive filters for real-time analysis
Tools & Skills Used:
MS Excel | Pivot Tables | Pivot Charts | Advanced Formulas | Dashboard Design | Data Analysis | Business Insights
Outcome:
Enabled stakeholders to quickly identify top-performing products, profitable regions, and growth trends, supporting better strategic decisions.
